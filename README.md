# 正平的博客

个人博客 — 时光不能停留但可以被记录

部署在 GitHub Pages，绑定域名 dscasa.com

## 结构

```
├── index.html          # 首页（文章列表）
├── about/index.html    # 关于页面
├── posts/              # 文章目录
│   └── hello/          # 第一篇文章
├── assets/css/         # 样式
└── rss.xml             # RSS 订阅
```

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 部署

推送到 GitHub 的 `wangyangywl.github.io` 仓库即可自动部署。
